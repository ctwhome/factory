ABC English mobile app
=========

An application to learn English with ABC School. 


Version
----

0.1 Alpha

Tech
-----------

This application uses a number of open source projects to work properly:

* AngularJS + RequireJS
* Ionic Framework
* Cordova
* Couchbase
--
* Modernizr
* Grunt
* [Ace Editor] - awesome web-based text editor

Run
--------------

```sh
grunt serve
```
